# 管理员 API

所有管理员接口需要 `👑` 权限。

---

## 1. 用户列表

```
GET /api/admin/users/
👑
```

**查询参数**:

| 参数 | 类型 | 说明 |
|---|---|---|
| `search` | string | 按用户名搜索 |
| `page` | int | 页码，默认 1 |
| `page_size` | int | 每页数量，默认 20 |

**响应**:
```json
{
    "count": 10,
    "results": [
        {
            "id": "uuid",
            "username": "user1",
            "email": "user1@example.com",
            "role": "user",
            "is_active": true,
            "date_joined": "2026-01-01T00:00:00"
        }
    ]
}
```

---

## 2. 切换用户状态

```
POST /api/admin/users/{user_id}/toggle/
👑
```

启用/禁用指定用户。

**响应**:
```json
{
    "id": "uuid",
    "username": "user1",
    "is_active": false
}
```

---

## 3. 重置用户密码

```
POST /api/admin/users/{user_id}/reset-password/
👑
```

生成 12 位随机新密码并重置。

**响应**:
```json
{
    "id": "uuid",
    "username": "user1",
    "new_password": "aB3xK9mQ2pL7"
}
```

---

## 4. 系统统计

```
GET /api/admin/stats/
👑
```

**响应**:

```json
{
    "user_count": 10,
    "fund_count": 12345,
    "position_count": 50,
    "nav_history_count": 100000,
    "latest_estimate_time": "2026-06-18T14:30:00",
    "version": "2.6.0"

```

---

## 5. 手动触发 Celery 任务

```
POST /api/admin/tasks/{task_name}/
👑
```

| task_name | 说明 |
|---|---|
| `update_fund_nav` | 同步基金历史净值（从数据源批量更新） |
| `update_fund_today_nav` | 同步当日确认净值 |
| `recalculate_positions` | 重算全部持仓汇总 |

**响应 (异步任务)**:
```json
{
    "status": "triggered",
    "task_name": "update_fund_nav",
    "task_id": "celery-task-uuid"
}
```

**响应 (同步任务 — recalculate_positions)**:
```json
{
    "status": "completed",
    "task_name": "recalculate_positions"
}
```

> Celery 不可用时返回 503。
