# Fundval API 文档

## 概述

Fundval 盘中基金实时估值与逻辑审计系统 RESTful API。

## 基础信息

- **Base URL**: `http://localhost:8000/api`（直接后端）/ `http://localhost:21345/api`（Nginx 代理）
- **协议**: HTTP/HTTPS
- **数据格式**: JSON
- **字符编码**: UTF-8
- **认证方式**: JWT Bearer Token（`Authorization: Bearer <token>`）
- **Token 有效期**: Access Token 1 小时 / Refresh Token 7 天

## 认证标记

| 标记 | 含义 |
|---|---|
| 🔓 | 无需认证 |
| 🔒 | 需要认证 |
| 👑 | 需要管理员权限 |

## HTTP 状态码

| 状态码 | 说明 |
|---|---|
| 200 | 成功 |
| 201 | 创建成功 |
| 204 | 删除成功（无响应体） |
| 400 | 请求参数错误 |
| 401 | 未认证 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 410 | 已初始化（Bootstrap） |
| 500 | 服务器错误 |
| 502 | AI 接口调用失败 |
| 503 | Celery 服务不可用 |
