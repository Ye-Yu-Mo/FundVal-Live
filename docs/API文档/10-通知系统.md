# 通知系统 API

## 1. 通知渠道

### 1.1 渠道列表

```
GET /api/notification-channels/
🔒
```

### 1.2 创建渠道

```
POST /api/notification-channels/
🔒
```

**请求体 (Webhook)**:
```json
{
    "channel_type": "webhook",
    "config": {
        "webhook_url": "https://hooks.example.com/xxx"
    },
    "is_active": true
}
```

自动识别飞书/钉钉/企业微信格式。

**请求体 (Email)**:
```json
{
    "channel_type": "email",
    "config": {
        "smtp_host": "smtp.qq.com",
        "smtp_port": 465,
        "use_ssl": true,
        "username": "user@qq.com",
        "password": "授权码",
        "to_email": "receiver@example.com"
    },
    "is_active": true
}
```

### 1.3 更新渠道

```
PATCH /api/notification-channels/{id}/
🔒
```

### 1.4 删除渠道

```
DELETE /api/notification-channels/{id}/
🔒
```

### 1.5 测试渠道

```
POST /api/notification-channels/{id}/test/
🔒
```

发送测试通知，验证渠道配置是否正确。

**响应**:
```json
{
    "message": "测试通知发送成功"
}
```

## 2. 通知规则

### 2.1 规则列表

```
GET /api/notification-rules/
🔒
```

**响应**:
```json
[
    {
        "id": "uuid",
        "fund": "fund-uuid",
        "fund_code": "000001",
        "fund_name": "华夏成长混合",
        "rule_type": "growth_up",
        "threshold": "5.00",
        "channels": [...],
        "is_active": true,
        "cooldown_minutes": 60,
        "created_at": "..."
    }
]
```

### 2.2 创建规则

```
POST /api/notification-rules/
🔒
```

**请求体**:
```json
{
    "fund": "fund-uuid",
    "rule_type": "growth_up",
    "threshold": "5.00",
    "channel_ids": ["uuid1", "uuid2"],
    "is_active": true,
    "cooldown_minutes": 60
}
```

| 字段 | 说明 |
|---|---|
| `rule_type` | `growth_up`(涨幅超过) / `growth_down`(跌幅超过) |
| `threshold` | 涨跌幅阈值（百分比，如 `5.00` 表示 5%） |
| `channel_ids` | 关联的通知渠道 UUID 列表 |
| `cooldown_minutes` | 冷却时间（分钟），防止重复通知 |

### 2.3 更新规则

```
PATCH /api/notification-rules/{id}/
🔒
```

### 2.4 删除规则

```
DELETE /api/notification-rules/{id}/
🔒
```

## 3. 通知记录

```
GET /api/notification-logs/
🔒
```

**查询参数**:

| 参数 | 类型 | 说明 |
|---|---|---|
| `rule_id` | UUID | 按规则过滤 |

**响应**:
```json
[
    {
        "id": "uuid",
        "fund_code": "000001",
        "fund_name": "华夏成长混合",
        "growth": "5.23",
        "status": "success",
        "error_message": null,
        "trigger_time": "2026-06-18T14:30:00",
        "channel_type": "webhook",
        "rule_type": "growth_up"
    }
]
```

## 定时检查

Celery Beat 每 5 分钟执行 `check_notification_rules`：遍历所有激活规则，检查基金估值涨跌幅是否触发阈值，考虑冷却时间，逐渠道发送通知。
