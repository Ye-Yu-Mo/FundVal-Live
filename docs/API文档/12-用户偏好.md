# 用户偏好 API

## 1. 获取偏好

```
GET /api/preferences/
🔒
```

无记录时返回默认值。

**响应**:
```json
{
    "preferred_source": "eastmoney",
    "theme_mode": "light",
    "report_enabled": false,
    "report_frequency": "monthly"
}
```

---

## 2. 更新偏好

```
PUT /api/preferences/
🔒
```

所有字段可选，独立更新（不传则不修改）。首次调用自动创建记录。

**请求体**:
```json
{
    "preferred_source": "danjuan",
    "theme_mode": "dark",
    "report_enabled": true,
    "report_frequency": "weekly,monthly"
}
```

| 字段 | 可选值 | 说明 |
|---|---|---|
| `preferred_source` | `eastmoney` / `yangjibao` / `xiaobeiyangji` / `danjuan` | 默认数据源。选择 danjuan 时估值自动 fallback 到 eastmoney |
| `theme_mode` | `light` / `dark` | 主题模式 |
| `report_enabled` | boolean | 是否开启定时投资报告 |
| `report_frequency` | 逗号分隔字符串 | `weekly`, `monthly`, `yearly` 可多选 |

**响应**:
```json
{
    "preferred_source": "danjuan",
    "theme_mode": "dark",
    "report_enabled": true,
    "report_frequency": "weekly,monthly"
}
```

---

## 偏好影响范围

| 偏好 | 影响 |
|---|---|
| `preferred_source` | 基金估值查询、自选列表估值、持仓预估市值、账户盈亏 |
| `theme_mode` | 全局 UI 主题（Ant Design darkAlgorithm） |
| `report_enabled` | 是否由 Celery Beat 定时生成 AI 投资报告 |
| `report_frequency` | 报告生成频率：周一(weekly) / 1日(monthly) / 1月1日(yearly) |
