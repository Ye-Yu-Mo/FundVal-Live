## 3. 二维码登录（养基宝）

### 3.1 获取二维码

```
POST /api/source-credentials/qrcode/
🔒
```

**请求体**:
```json
{
    "source_name": "yangjibao"
}
```

**响应**:
```json
{
    "qr_id": "qr-xxx",
    "qr_url": "http://..."
}
```

### 3.2 检查扫码状态

```
GET /api/source-credentials/qrcode/{qr_id}/state/?source_name=yangjibao
🔒
```

**响应**:
```json
{
    "state": "confirmed",
    "token": "xxx"
}
```

| state | 说明 |
|---|---|
| `waiting` | 等待扫码 |
| `scanned` | 已扫码，待确认 |
| `confirmed` | 已确认（后端自动保存凭证） |
| `expired` | 已过期 |

## 4. 手机号登录（小倍养基）

### 4.1 发送验证码

```
POST /api/source-credentials/phone/send-sms/
🔒
```

**请求体**:
```json
{
    "source_name": "xiaobeiyangji",
    "phone": "13800138000"
}
```

### 4.2 验证登录

```
POST /api/source-credentials/phone/verify/
🔒
```

**请求体**:
```json
{
    "source_name": "xiaobeiyangji",
    "phone": "13800138000",
    "code": "123456"
}
```

**响应**:
```json
{
    "message": "登录成功",
    "source_name": "xiaobeiyangji"
}
```

登录成功后自动保存凭证。

## 5. 登录状态查询

```
GET /api/source-credentials/status/?source_name=yangjibao
🔒
```

**响应**:
```json
{
    "logged_in": true,
    "login_type": "qrcode",
    "source_name": "yangjibao",
    "is_active": true,
    "created_at": "2026-01-01T00:00:00"
}
```

`login_type`: `none` / `qrcode` / `phone`。未登录时返回 `logged_in: false`。

凭证查找策略：当前用户 → 任意用户（fallback），管理员配置后全局可用。

## 7. 导入持仓

```
POST /api/source-credentials/import/
🔒
```

**请求体**:
```json
{
    "source_name": "yangjibao",
    "overwrite": false
}
```

| 字段 | 说明 |
|---|---|
| `source_name` | `yangjibao` / `xiaobeiyangji`，默认 `yangjibao` |
| `overwrite` | `true` 清空已有持仓流水后重新导入 |

**响应**:
```json
{
    "accounts_created": 2,
    "accounts_skipped": 1,
    "holdings_created": 5,
    "holdings_skipped": 3
}
```

**导入逻辑**:
- 养基宝: 创建父账户「养基宝」，按券商/平台创建子账户
- 小倍养基: 创建父账户「小倍养基」，按 accountId 分组创建子账户
- 每个持仓生成一条 BUY 操作流水，自动触发 `recalculate_position`
- `amount = share × nav`（买入成本，非当前市值）

**限制**: 必须已登录对应数据源。

## 净值同步链路

`sync_nav_history` 的多源 fallback 顺序：

```
EastMoneySource (Web: pingzhongdata JSONP)
  └─ 空 → EastMoneySource (Mobile: FundMNHisNetList JSON)
       └─ 仍空 → DanjuanSource (danjuanfunds.com)
            └─ 仍空 → 返回 count=0
```

QDII 基金（净值 T+2 延迟）: `fetch_estimate` 检测到 fundgz 基准净值过期时，自动从 Mobile API 取最新净值重新计算涨跌幅。
