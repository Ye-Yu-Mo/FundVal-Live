# 基金管理 API

## 1. 基金列表

```
GET /api/funds/
🔓
```

**查询参数**:

| 参数 | 类型 | 默认 | 说明 |
|---|---|---|---|
| `search` | string | — | 按代码/名称搜索 |
| `fund_type` | string | — | 按类型过滤 |
| `page` | int | 1 | 页码 |
| `page_size` | int | 20 | 每页数量 |

**响应**:
```json
{
    "count": 12345,
    "results": [
        {
            "id": "uuid",
            "fund_code": "000001",
            "fund_name": "华夏成长混合",
            "fund_type": "混合型-灵活",
            "latest_nav": "1.4070",
            "latest_nav_date": "2026-06-17",
            "created_at": "2026-01-01T00:00:00Z",
            "updated_at": "2026-01-01T00:00:00Z"
        }
    ]
}
```

## 3. 获取基金估值

```
GET /api/funds/{fund_code}/estimate/
🔓
```

**查询参数**:

| 参数 | 类型 | 默认 | 说明 |
|---|---|---|---|
| `source` | string | `eastmoney` | `eastmoney` / `yangjibao` / `xiaobeiyangji` / `danjuan` |

> `source=danjuan` 时静默 fallback 到 eastmoney（蛋卷无实时估值）。QDII 基金自动校正过期净值基准。

**响应**:
```json
{
    "fund_code": "000001",
    "fund_name": "华夏成长混合",
    "estimate_nav": 1.4100,
    "estimate_growth": 0.32,
    "estimate_time": "2026-06-18T14:30:00"
}
```

---

## 4. 批量获取基金估值

```
POST /api/funds/batch_estimate/
🔓
```

**请求体**:
```json
{
    "fund_codes": ["000001", "161725"],
    "source": "eastmoney"
}
```

**响应**: 以基金代码为 key 的字典对象。缓存命中时 `from_cache: true`。

```json
{
    "000001": {
        "fund_code": "000001",
        "fund_name": "华夏成长混合",
        "estimate_nav": "1.4100",
        "estimate_growth": "0.32",
        "estimate_time": "2026-06-18T14:30:00",
        "latest_nav": "1.4070",
        "latest_nav_date": "2026-06-17",
        "from_cache": true
    },
    "999999": {
        "fund_code": "999999",
        "error": "基金不存在"
    }
}
```

**缓存机制**: TTL 默认 5 分钟，通过 `config.json` 的 `estimate_cache_ttl` 配置。最多 5 线程并发请求外部 API。

## 5. 批量更新基金净值

```
POST /api/funds/batch_update_nav/
🔓
```

**请求体**:
```json
{
    "fund_codes": ["000001", "161725"]
}
```

**响应**:
```json
{
    "000001": {
        "fund_code": "000001",
        "latest_nav": "1.4070",
        "latest_nav_date": "2026-06-17"
    }
}
```

不会覆盖更新日期的净值（只写入 `new_date >= latest_nav_date`）。

## 7. 查询持仓操作净值

```
POST /api/funds/query_nav/
🔒
```

建仓/加仓时自动查询操作日期对应的基金净值。

**请求体**:
```json
{
    "fund_code": "000001",
    "operation_date": "2024-01-15",
    "before_15": true
}
```

| 字段 | 说明 |
|---|---|
| `before_15` | `true`=15:00前操作 → 查 T-1 净值；`false`=15:00后 → 查 T 日净值 |

**响应**:
```json
{
    "fund_code": "000001",
    "fund_name": "华夏成长混合",
    "nav": "1.2345",
    "nav_date": "2024-01-14",
    "source": "history"
}
```

| source | 说明 |
|---|---|
| `history` | 从 FundNavHistory 表查到 |
| `synced` | 实时从数据源同步后查到 |
| `latest` | fallback 到 Fund.latest_nav |

自动处理非交易日（往前找最近交易日），无历史数据时尝试实时同步。

```json
            "change_percent": "1.23",
            "contribution": "0.2064"
        }

```

## 11. 基金估��准确率

```
GET /api/funds/{fund_code}/accuracy/
🔓
```

**查询参数**:

| 参数 | 类型 | 默认 | 说明 |
|---|---|---|---|
| `days` | int | 100 | 统计天数 |

**响应**: 按数据源分组。

```json
{
    "eastmoney": {
        "records": [
            {
                "date": "2026-06-17",
                "estimate_nav": "1.4070",
                "actual_nav": "1.4050",
                "error_rate": "0.001423"
            }
        ],
        "avg_error_rate": "0.0012",
        "record_count": 30
    }
}
```

误差率 = `(估值 - 实际) / 实际`（正数=高估，负数=低估）。

## 12. 盘中估值曲线

```
GET /api/funds/{fund_code}/estimate-intraday/
🔓
```

**查询参数**:

| 参数 | 类型 | 默认 | 说明 |
|---|---|---|---|
| `source` | string | `eastmoney` | `all` 返回所有数据源 |

**响应**:
```json
{
    "fund_code": "000001",
    "snapshots": [
        {
            "source": "eastmoney",
            "timestamp": "2026-06-18T09:35:00",
            "estimate_nav": "1.4080",
            "estimate_growth": "0.07"
        }
    ]

```

## 14. 基金排行榜

```
GET /api/funds/rankings/
🔓
```

**查询参数**:

| 参数 | 类型 | 默认 | 说明 |
|---|---|---|---|
| `type` | string | `gain` | `gain`(涨幅榜) / `popular`(人气榜) / `accuracy`(估值准度榜) |
| `category` | string | — | 按类型过滤（如 `股票型`） |
| `page` | int | 1 | 页码 |

**响应**:
```json
{
    "count": 100,
    "results": [
        {
            "fund_code": "006502",
            "fund_name": "财通集成电路产业股票A",
            "fund_type": "股票型",
            "latest_nav": "9.9436",
            "estimate_growth": "61.18"
        }
    ]
}
```

人气榜额外返回 `pos_count` 字段，准度榜返回 `avg_error` 字段。

## 15. 基金 PK 对比

```
GET /api/funds/compare/
🔓
```

**查询参数**:

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `codes` | string | 是 | 基金代码，逗号分隔（2-5 只） |

**响应**:
```json
{
    "funds": [
        {
            "fund_code": "000001",
            "fund_name": "华夏成长混合",
            "fund_type": "混合型-灵活",
            "latest_nav": "1.4070",
            "returns": {
                "1m": "5.23",
                "3m": "12.45",
                "6m": "8.90",
                "1y": "15.67"
            },
            "metrics": {
                "max_drawdown": "-25.30",
                "volatility": "18.50",
                "sharpe": "0.85"
            }
        }
    ]
}
```

`metrics` 需要基金有 ≥60 条净值记录才能计算。夏普比率使用 2% 无风险利率。

## 16. 基金详情（评级排名）

```
GET /api/funds/{fund_code}/fund_detail/
🔒
```

> 仅 danjuan 数据源支持。

**查询参数**:

| 参数 | 类型 | 默认 | 说明 |
|---|---|---|---|
| `source` | string | `danjuan` | 仅支持 `danjuan` |

**响应**:
```json
{
    "fund_code": "000001",
    "source": "danjuan",
    "detail": {
        "fund_name": "华夏成长混合",
        "fund_type": "3",
        "risk_level": "4",
        "manager_name": "刘睿聪 郑晓辉",
        "company_name": "华夏基金管理有限公司",
        "latest_nav": "1.4070",
        "nav_date": "2026-06-17",
        "period_returns": {
            "1m": "12.92", "3m": "32.36", "6m": "33.87",
            "1y": "72.32", "3y": "55.52", "5y": "5.58"
        },
        "peer_ranking": {
            "1m": "995/5347", "3m": "1129/5227", "1y": "1263/4588"
        }
    }
}
```

`fund_type` 蛋卷编码: 1=股票型 2=债券型 3=混合型 5=指数型 11=QDII。
